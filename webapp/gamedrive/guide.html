Welcome to GameDrive: Your Local Save Manager

GameDrive is designed to provide a secure and simple way to manage your game saves across devices. This guide will walk you through the core features and, most importantly, explain how we protect your data and what steps you can take for ultimate peace of mind.

1. Managing Your Games

Adding a New Game

  1. Navigate to the main Dashboard.
  2. Click the Add New Game button.
  3. Enter the Game's Title and any identifying notes.
  4. The game now appears in your list, ready to store saves!

Uploading a Save File

  1. Select the game from your Dashboard list.
  2. Click Upload New Save.
  3. Browse for the save file (usually a single file or a zip archive of a save folder).
  4. Add a descriptive label (e.g., "Endgame before boss fight") and click Upload. Your save is now secure in the cloud.

2. Restoring Saves and Preventing Data Loss

Restoring a Backup (Download)

To restore your game progression, simply locate the save file in your game's history on the dashboard.

  1. Select the game, then find the desired save in the list.
  2. Click the Download icon next to the save entry.
  3. Once downloaded, move the file(s) into your game's local save directory, overwriting any existing files.
  4. Also note that even if you delete these save files on your computer they will still be downloadable through the backed up save file, The only way you can lose it is by clearing your browser data without having the website installed as a app on ios or windows

Data Loss Prevention Strategy

Your data is primarily protected by being stored securely in your browsers local storage or (IndexDB), With this being said you need to make sure your backing up your data and keeping it somewhere safe.

  • Cross-Device Access: Since GameDrive is a StadiaWebapp, your backups are accessible anywhere you import them, even if you dont have the files installed on that device. Importing a backup will let you download them

Note: Always keep a recent manual export file saved in a separate location for complete peace of mind.

3. Q&A

How do I delete an old save?
Select the game, find the save in the list, and click the trash can icon. Note that this action is immediate and cannot be undone!
What if my game's save data is a folder, not a single file?
You must compress the folder into a single .zip file before uploading to GameDrive. When restoring, simply unzip the file to your game's save location.
Is my data secure?
Yes. All save files are stored securely in a protected local environment, This means whatever is on your directory is only accessible for you.
Can i delete the files once their in the drive?
Also Yes, Because their uploaded to your indexDB. But however backing up the file will equal out the same amount of storage thats why you have to back it up somewhere else as well.
How do i download it as a app and what are the benefits?
